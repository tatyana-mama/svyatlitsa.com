Smart Clinic Infrastructure

Aqara IoT ecosystem

Complete smart building system for 7 dental offices. Cameras, temperature sensors, motion sensors, door sensors, smart plugs, and Zigbee hubs. Managed via Home Assistant on NVIDIA Jetson AGX Orin.

System Architecture
Per-Office Configuration
Each of the 7 dental offices is equipped with an identical set of Aqara devices connected via Zigbee 3.0. Central management through Home Assistant with ONVIF camera integration.
7 Identical Offices
Standardized IoT kit per office: 1 hub, 1 camera, 3 temp sensors, 1 motion sensor, 1 door sensor, smart plugs for all equipment.
3 WiFi Networks
CORAL_REEF (staff), svyatlitsa (patients), svyatlitsa_iot (dedicated IoT network for Aqara devices).
Home Assistant + Jetson
All devices managed via Home Assistant on NVIDIA Jetson AGX Orin 64GB. Cameras via ONVIF, sensors/plugs via Zigbee.
Local Processing
All automations run locally without cloud dependency. Camera AI detection processed on-device. Full privacy compliance.
